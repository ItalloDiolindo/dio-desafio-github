TCP/IP, PORTAS, ROTEADORES, SWITCHES E MODEMS

Tcp/Ip - Protocolos de comunicação entre computadores
Modelo de camadas - 4 camadas
	1 - Física (placa de rede)
	2 - Rede (IP)
	3 - Transporte (TCP ou UDP)
	4 - Aplicação (FTP, SMTP - email, HTTP - internet)

TCP x UDP

	UDP - Rápido, não confiável, carro do ovo e Livestream

	TCP - Voltado à conexão, handshake, integridade e ordem dos dados, aplicativo de mensagens de texto

ROTEADORES, SWITCHES E MODEMS

Modem = Modulo e demodula sinais, converte dados em um formato que possa ser transmitido de um pc para o outro e lido por outro

Rotedor = ditribui internet para um ou mais dispositivos, pode fazer comunicação entre redes

Switch = distribui internet para um ou mais dispositivos de uma rede e são inteligentes

Quais portas usadas pelo MySQL e pelo Apache?
	Apache = Porta 443
	MySQL = Porta 3306

BROWSER, SITES, APLICATIVOS E WEBSERVER

Browser = Sabe identificar várias linguagens de programação, de marcação (html) e conteúdo multimedia, existem plug-ins ou add-ons. 

Site = Página da internet, diversos propósitos, podem ser feitas em diversas linguagens

Aplicativo = Software que é executado no navegador

E-commerce = Comércio eletrônico, compra e venda com sistema de pagamento

Web-server = Estático (servidor físico) e o dinâmico (softwares que estão no servidor físico)

Dinamico - arquivos (file server), aplicações (application server), banco de dados (database)

Web-service - Interface disponível para fazer requisições e consultas em bancos de dados inacessíveis (Correios, Governo)
__________________________________________________
STACKS

Stacks = Pilhas de tecnologia, softwares necessários e suficientes para executar um aplicativo, linguagens de programação, ambientes e ferramentas de interação com app/sw, capacidade e limitação de performance, pontos fortes e fracos do app/sw.

Front-end = parte da frente (site, software, aplicativo, web service), interface (UI e UX), lógica de programação (HTML, CSS, jQuery, PHP)

Back-end = parte trás, servidores, banco de dados, meio campo entre interface e bancos de dados, regra de negócios, validações, MySQL, Oracle, Java, Node.js

Full stack = sabe trabalhar em todas as camadas das tecnologias de desenvolvimento/execução de um app/sw
